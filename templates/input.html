Relational Database Normalization

{% raw %} Input must be of the form:
<Line> ---> <Relation Name>[<Tuples>];
<Tuples> ---> <Tuple> | <Tuple>,<Tuples>
<Tuple> ---> (< attrbute name >, < isPrimaryKey >, < isNULL >, < isForeignKey >, < isKey >)
with each line corresponding to schema of one relation in the database {% endraw %}
{% raw %} Input consits of several lines, each specifying FDs for a relation in the schema. Input format for each line:
<Line> ---> <Fds><Fd>
<Fds> ---> empty | <Fd>;<Fds>
<Fd> ---> {<Lattributes>}->{<Ratttributes>}
<Lattributes> ---> <attrbute_name> | <attrbute_name>&<Lattributes>
<Ratttributes> ---> <attribute_name> | <attribute_name>,<Ratttributes>
{% endraw %}
